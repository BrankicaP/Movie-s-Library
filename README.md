# Movie-s-Library - Flask app

## Create **venv**
### Windows
    python -m venv venv
### Linux/MacOS
    python -m venv venv

## Activate **venv** 
### Windows
    venv\Scripts\activate
### Linux/MacOS
    source venv/bin/activate

## Install requirements
    pip install -r requirements.txt

## Set FLASK_APP
    set FLASK_APP=app.py

## Run app 
    flask run
